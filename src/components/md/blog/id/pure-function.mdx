Kalau belum, pure function adalah fungsi yang selalu menghasilkan output yang sama jika input-nya sama dan tidak memiliki **side effect**.

Artinya, fungsi ini:

- Tidak mengubah data di luar scope-nya sendiri,
- Tidak mengubah parameter yang diterimanya,
- Tidak bergantung pada data eksternal lain yang dapat berubah-ubah.

## âœ¨ Karakteristik Pure Function

- **Deterministik**: Output hanya bergantung pada input. Jadi, jika input-nya sama, maka output-nya pasti sama juga.
- **Tanpa Side Effect**: Tidak melakukan perubahan pada variabel global, tidak mengubah nilai parameter, dan tidak berinteraksi dengan elemen luar seperti database atau filesystem.

Pure function membantu membuat kode lebih mudah diprediksi dan diuji. Konsep ini penting dalam pemrograman fungsional dan menjadi dasar dalam pengembangan perangkat lunak modern.
