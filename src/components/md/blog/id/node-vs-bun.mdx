Node.js dan Bun.js adalah runtime JavaScript yang dapat digunakan untuk membuat aplikasi server-side. Node.js sudah lama menjadi standar industri dan memiliki ekosistem yang sangat matang, sedangkan Bun.js adalah pendatang baru yang memiliki banyak sekali tools, seperti runtime, package manager, dan testing tools yang sudah terinklusi di dalamnya.

- [Node.js](https://nodejs.org/id)
- [Bun.js](https://bun.sh/docs)
