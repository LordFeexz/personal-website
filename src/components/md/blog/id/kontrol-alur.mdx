Pernah nggak sih saat lagi ngoding, terus kamu lihat kode sendiri terasa rumit dan nggak “estetik”?

Mungkin, kamu perlu menyederhanakan kode agar lebih efektif dengan control flow yang lebih baik.

## Apa Itu Control Flow?

Control flow adalah cara program menentukan alur eksekusi berdasarkan kondisi tertentu. Ada tiga konsep utama dalam control flow:

- **Percabangan:** Menjalankan baris kode tertentu berdasarkan kondisi. Umumnya menggunakan kata kunci `if` `else` `switch`
- **Perulangan:** Menjalankan baris kode yang sama secara berulang selama kondisinya terpenuhi. Umumnya menggunakan kata kunci `for` `while`
- **Pemrosesan Berurutan:** Alur eksekusi kode dari atas ke bawah tanpa percabangan atau perulangan.

### Contoh Control Flow Sehari-hari

```js
if (cuaca === "hujan") {
  pakaiPayung();
}
pergiKeluar();
```

## Kenapa ini penting?

Karna dengan control flow yang baik maka kita dapat:

- **Mengelola alur program dengan lebih efisien.**
- **Menghindari pengulangan kode yang tidak perlu.**
- **Membuat logika yang lebih fleksibel dan dinamis.**
